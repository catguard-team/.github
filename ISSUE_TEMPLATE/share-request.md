> 막히면 디스코드 [#도와줘](https://discord.gg/Dp4pKwns2Y) 채널에 한 줄 던지세요.
> 모르는 항목은 비워두고 보내셔도 됩니다. 단장이 같이 봐드립니다.
>
> **참고**: 자경단은 자체 레포 외에는 **사본을 두지 않습니다**. 다른 사람 레포는 링크로만 모읍니다(모으는 역할만). 즉 본인 레포는 본인 영역에 그대로 두고, 자경단 인덱스에 링크만 등재됩니다.

## 1. 내 정보

- **GitHub 사용자명**: @여기에이름
- **공유하려는 레포 링크**: `https://github.com/...`
- **자경단원 디스코드 닉**: (디스코드 합류 안 하셨으면 생략)

---

## 2. 이 자료가 뭔가요?

> 한 줄로. 길게 안 써도 됩니다. 예: "초등학생용 파이썬 게임 만들기 강의 자료"

(여기에 한 줄)

---

## 3. 어떤 종류인가요?

- [ ] 강의/튜토리얼 (영상, 슬라이드, 대본 등)
- [ ] 코드/도구 (라이브러리, 스크립트, 앱)
- [ ] 글/문서 (블로그 포스트, 가이드)
- [ ] 디자인/에셋 (이미지, 폰트, 아이콘)
- [ ] 기타: ___

---

## 4. 라이선스가 있나요?

레포 안에 `LICENSE` 파일이 있는지 한 번만 확인해주세요.

- [ ] **있음** → 종류: ___ (예: MIT, Apache 2.0, CC BY 4.0)
- [ ] **없음** → 단장이 도와드립니다. 추천: MIT(코드) / CC BY 4.0(글·이미지)

> 모르겠으면 "없음" 체크하고 그냥 보내세요. 자경단이 같이 정리해드립니다.

---

## 5. 자가 위험 점검 (제일 중요)

아래 7가지가 본인 자료와 상관없는지 한 번씩 떠올려보세요. 7개 다 통과해야 자경단에 등재됩니다. **자신 없는 항목은 비워두고 보내셔도 OK** — 단장이 같이 봐드립니다.

- [ ] 어린이를 위험에 빠뜨릴 수 있는 내용이 없습니다 (그루밍, 부적절한 만남 유도 등)
- [ ] 노인·취약계층을 속이는 사기 수법이 없습니다 (가짜 보이스피싱 도구 등)
- [ ] 자해·자살을 부추기는 내용이 없습니다
- [ ] 특정 집단을 차별·혐오하는 표현이 없습니다 (성별·인종·종교·장애 등)
- [ ] API 키, 비밀번호, 다른 사람의 개인정보가 코드에 들어 있지 않습니다
- [ ] 다른 사람의 저작물을 허락 없이 가져온 게 없습니다 (이미지, 글, 코드 모두)
- [ ] 특정 인물을 명예훼손하는 표현이 없습니다

> **거절돼도 자격 박탈 같은 거 없습니다.** 그냥 같이 고쳐서 다시 올리면 됩니다. 자경단의 검증은 위험만 봅니다 — 톤·문체·디자인은 보지 않습니다(3대 존중 원칙).

## 다음에 일어날 일

1. **단장이 24~72시간 안에 답합니다.** 늦어져도 잊은 거 아닙니다. 미리 사과드립니다.
2. 위험 점검 7개를 단장이 한 번 더 같이 봅니다. (다시 한번 강조: 톤·문체는 안 봅니다.)
3. 통과되면 단장이 자경단 인덱스(`.github/profile/README.md` "자경단원 강의·자료" 섹션 등)에 **링크만** 추가합니다. 본인 레포에 별도 셋업 필요 없습니다.
4. 디스코드 #공지 채널에 알려드리고, 검증 배지를 본인 README에 박을 수 있는 마크다운도 드립니다.

## 마음에 안 들면 언제든

- 등재된 후라도 언제든 빼달라고 하실 수 있습니다 (`remove-request` 템플릿).
- **즉시 빼기**가 기본입니다 (사본이 없으니 유예 기간 의미 없음).
- **저작권·유지보수권은 항상 원작자분 거예요.** 자경단은 사본을 두지 않습니다.
