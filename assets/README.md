## 어디에 어떤 버전?

| 위치 | 권장 |
|------|------|
| GitHub org 아바타 | `logo-mini.svg` (64×64 PNG로 export 추천) |
| .github/profile/README | `logo-main.svg` 또는 `logo-wordmark.svg` |
| 디스코드 서버 아이콘 | `logo-mini.svg` |
| 굿즈(스티커, 티셔츠) | `logo-badge.svg` |
| favicon | `logo-mini.svg` |
| 문서 푸터 | `logo-wordmark.svg` |

## 사용 규칙

- 색은 위 팔레트만 사용. 무지개 그라데이션 금지.
- 비율 유지. 가로로 늘리거나 세로로 누르지 말 것.
- 외부 단체와 협력 시 자경단 로고 단독 사용 금지(공동 표기 권장).
- 본인 사이드 프로젝트에 자경단 로고 박는 건 자유. 단, 자경단 공식 인증을 의미하지 않음.

## 정식 로고 작업이 필요하면

[`saenoyangi-handbook` 또는 `kkulzam-spot` 레포에 issue](https://github.com/catguard-team) 또는 디스코드 #그루밍 채널.

자경단원 누구든 PR 환영. 단, 다음은 거절 사유:

- 지구본 모티프
- 손잡기, 하트 모티프
- 활짝 웃는 표정
- 별·왕관·후광
- 클립아트 느낌의 귀여운 고양이

자경단의 시각 톤은 **시크함 + 자조 + 발자국**입니다. 귀여움보다 단단함.
